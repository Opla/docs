1. Les intents #Any

On va créer un intent #Any:

on clique sur créer un nouvel intent, on renomme l’intent en #Any, et on supprime les inputs et outputs déjà présents en cliquant sur le signe moins en bout de ligne pour input (marqué “intent”) et output (“I don’t understand”)

Puis on clique sur insert block any dans les inputs de #Any :

On clique sur le + bleu et sur SAVE ;

Nous allons nous intéresser au sujet des variables.

Une variable peut être gérée de deux façons différentes :

  • soit elle est discrète c’est-à-dire qu’elle ne s’affiche pas dans la fenêtre du playground ou du publish, c’est le cas si on l’écrit dans un bouton code code ,
  • soit elle est explicite et s’affiche dans la fenêtre du playground ou du publish, c’est le cas si on l’écrit dans le bouton output code assignment.

La mention @any dans l’intent #Any sert à recueillir des informations non contenues dans les autres intents, en l’occurrence une valeur saisie librement par l’utilisateur. Pour capter cette valeur, et l’enregistrer dans une variable de nom “valeur” par exemple, il faut faire la manipulation suivante : cliquer sur code, code et remplir le cadre résultant avec la déclaration du nom de la variable, le nom “valeur”, et les symboles suivants, =* , qui veulent dire assigner une valeur à la variable valeur, exemple :

#

Output:
code
valeur=*

2. Les boucles sur Non

On va enfin ajouter les deux boucles “Non” : on remarque que la flèche les fait remonter d’une étape : on va donc prendre comme conditionnelle la ligne de code précédente, ce qui va faire reboucler le chatbot sur la question.

On va écrire le “Non” en reprenant et inversant les actions saisirtelephone et confirmertelephone, on va mettre confirmertelephone dans la conditionnelle et saisirtelephone dans le code.

#Non

Output:
action = confirmertelephone
Quel est votre numéro de téléphone? code
action=saisirtelephone

De même, sur mail on va écrire un “Non” entre confirmermail et saisirmail

#Non

Input:
non
Output:
action = confirmermail
Quel est votre mail ? code
action=saisirmail
